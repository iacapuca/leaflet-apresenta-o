Vamos falar sobre tiles!
Raster tiles, dessa vez. Vector tiles talvez em outra oportunidade.
"... web maps são feitos de varias imagens quadradas chamadas de tiles."
"Cada tile costuma ter 256x256 pixels e são posicionados lado a lado a fim de criar a ilusão de uma imagem muito grande sem emendas."
Esses pequenos tiles carregam bem mais rapido que um grande mapa.
Esse tipo de mapa costuma ser chamado de slippy map.
Cada nivel de zoom possui o seu proprio conjunto de tiles.
O nivel 0: 1 tile para o mundo inteiro.
Para cada nivel adicional, o número de tiles cresce exponencialmente.
Nivel 1: 4 tiles para o mundo inteiro.
Nivel 2: 16 tiles para o mundo inteiro.
... acho que vocês pegaram a ideia.
Os tiles só renderizam quando você está olhando para eles, nesse momento eles formam mapas... bem rapido!.
Como você percebeu, tiles são apenas imagens estaticas na internet.
Massa. Mas o que isso te a ver com Leaflet?
Esses tiles precisam ganhar vida de alguma forma numa página da web. Eles precisam saber quando carregar e como reagir quando você clicar ou arrastar.
Leaflet é uma biblioteca de JavaScript que nos permite fazer isso.
Incluir bibliotecas no nosso codigo é como copiar e colar parte do codigo de outra pessoas no nosso.
Você tem acesso a tudo na biblioteca. No nosso caso é um cojunto de ferramentas para fazer web maps e dar a eles funcionalidade.
